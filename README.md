# 💹 Customer Churn Analysis

##  Project Overview

This project focuses on analyzing customer churn in a telecom company using Python in Jupyter Notebook. The objective is to identify key factors that contribute to customer churn and provide actionable insights to improve customer retention.


##  Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Jupyter Notebook

##  Data Cleaning & Preparation

* Handled missing values in `TotalCharges` by replacing blanks with 0 (for customers with zero tenure)
* Converted categorical values (e.g., Senior Citizen from 0/1 to Yes/No) for better interpretability
* Ensured data consistency and proper formatting for analysis


## ➡️ Exploratory Data Analysis (EDA)

Performed detailed analysis to understand customer behavior and churn patterns using visualizations and summary statistics.


## Key Insights

*  **Overall Churn Rate:** Around **26.5%** of customers have churned
*  **Senior Citizens:** Higher churn rate compared to non-senior customers
*  **Tenure Impact:** Customers with shorter tenure (1–2 months) are more likely to churn
*  **Contract Type:** Month-to-month contract users have the highest churn rate
*  **Service Usage:**

      * Customers using services like **PhoneService, DSL Internet, and OnlineSecurity** tend to stay longer
      * Lack of services like **OnlineBackup, TechSupport, StreamingTV** increases churn likelihood
* 💳 **Payment Method:** Customers using **electronic check** show higher churn behavior
